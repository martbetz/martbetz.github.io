Installing RawTherapee

GNU/Linux

There are several different ways to download and install RawTherapee:

  • via the native repo – this should be considered the prefered method of installation for most users; repology.org provides an extensive index of the latest available RawTherapee distro packages
  • via the official RawTherapee website – rawtherapee.com distributes the latest RawTherapee AppImage directly via its 'downloads' page
  • as a third-party binary/package – flathub.org distributes the latest RawTherapee Flatpack
  • by compiling from source – rawpedia.org provides an extensive guide to compiling RawTherapee on Linux and related systems

macOS

There are several different ways to download and install RawTherapee:

  • via the official RawTherapee website – this should be considered the prefered method of installation for most users; rawtherapee.com distributes the latest RawTherapee macOS binary directly via its 'downloads' page
  • as a third-party binary/package – macports.org distributes the latest RawTherapee MacPort; formulae.brew.sh also distributes a RawTherapee Cask
  • by compiling from source – rawpedia.org provides an extensive guide to compiling RawTherapee on macOS

Microsoft Windows

There are several different ways to download and install RawTherapee:

  • via the official RawTherapee website – this should be considered the prefered method of installation for most users; rawtherapee.com distributes the latest RawTherapee Windows installer directly via its 'downloads' page
  • by compiling from source – rawpedia.org provides an extensive guide to compiling RawTherapee on Microsoft Windows
